# PrivacyScope

**你的 Windows 在偷偷往外传什么？**

Windows 默认会收集你的打字习惯、浏览历史、位置轨迹、甚至设备名称，通过遥测系统上传给 Microsoft。大多数用户不知道开关在哪，也不知道关了多少项。

PrivacyScope 给你一个**隐私体检分数**，告诉你哪些门还开着，并一键帮你关上。

---

## 能扫出什么？

| 类别 | 扫描内容 | 常见问题 |
|------|----------|----------|
| 遥测与数据收集 | 遥测级别、设备名称上报、活动历史 | 大多数机器默认为"完整"级别 |
| 广告与追踪 | 广告 ID、应用启动追踪、定制体验 | 跨应用行为追踪默认开启 |
| Cortana 与搜索 | 云端搜索、语音数据共享 | 本地搜索也会发往 Bing |
| 位置服务 | 系统定位、应用位置访问 | 即使不用地图，位置也在泄露 |
| 应用权限 | 开机自启项、后台运行 | 13+ 个启动项是常态 |
| 网络与共享 | WiFi Sense、P2P 更新共享 | 你的 WiFi 密码可能分享给了联系人 |

**典型结果：一台全新 Windows 11 机器，不做任何配置，能扫出 6-12 个隐私问题，得分 15-40/100。**

## 使用方法

```bash
# 安装依赖（仅需一次）
pip install pywin32

# 运行扫描（普通权限即可）
python main.py

# 如需一键修复，右键 → 以管理员身份运行
```
