# PrivacyScope - Windows 隐私仪表盘

一键扫描 Windows 隐私设置，发现并修复遥测、广告追踪、位置泄露等问题。

## 功能

- **隐私评分**: 100 分制，直观了解你的隐私状况
- **7 大类扫描**: 遥测、广告追踪、Cortana/搜索、位置、应用权限、网络共享、活动历史
- **一键修复**: 自动修复大部分隐私问题
- **全本地运行**: 不联网，不上传任何数据

## 安装与运行

```bash
# 需要 Python 3.8+，仅 Windows 10/11
pip install pywin32
python main.py

# 如需修复功能，请右键 → 以管理员身份运行
```

## 扫描项目

| 类别 | 检查项 |
|------|--------|
| 遥测 | 遥测级别、设备名称上报、活动历史 |
| 广告 | 广告 ID、应用启动追踪、量身定制体验 |
| Cortana | 云端搜索 |
| 位置 | 位置服务 |
| 启动 | 开机自启项数量 |
| 网络 | WiFi Sense 共享 |
